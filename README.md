## Deskripsi singkat:
Projek ini adalah website sederhana yang bisa mengkomparasi teknik atau metode Corner Detection pada suatu citra.
## List Metode Corner Detection yang digunakan:
  1. Hit and Miss
  2. Harris Corner Detection
  3. Shi-Tomasi
  4. Contour
